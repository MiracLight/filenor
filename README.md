# Filenor: Optimize Formatter

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![Python](https://img.shields.io/badge/Python-3.8%2B-brightgreen.svg)

**Filenor** is a high-performance file conversion and optimization tool that operates entirely on local resources, prioritizing user privacy. Thanks to its modern interface and portable structure, it processes your sensitive data securely without requiring an internet connection.

## 📋 About the Project
Filenor was developed to close the security gap caused by online converters that upload user data to third-party servers. All conversion processes take place on the user's own hardware.

### Key Features
*   **Document Processing:** Lossless conversion of MS Office documents to PDF.
*   **Media Conversion:** High-speed video/audio conversion and audio extraction via FFmpeg.
*   **Image Optimization:** Converting PDF pages to high-resolution images.
*   **Portable Architecture:** No installation or registry entries required; suitable for USB drives.
